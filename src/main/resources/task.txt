Рисование дерева
Пользователю доступно меню, посредством которого он может формировать бинарное дерево любого размера,
узлами которого являются числа. Соответственно можно добавлять, удалять, изменять узлы.
По запросу пользователя приложение должно рисовать это дерево в максимально компактном виде.

1. разбить классы по пакетам
2. вывод меню и рисование в отдельный классы (View)
3. добавить логирование: юзать log4j (https://habrahabr.ru/post/247647/), пользователю не должны падать вообще никакие ошибка,
 только предупреждения что ввели что-то неверно, все ошибки программы должны логироваться в файл
4. должна быть возможность создания графа с нуля
5. должна быть возможность добавить узел в конкретное место
6. добавь функцию балансировки дерева

7. Убрать статику
8. Переделать добавление
9. Удаление всего вниз по иерархии
10. Добовить добавление корня дерева
11. Сделать сервлетами
